# 100-line pong game written in Zig.

For my programming languages (CS341) course, I was assigned with researching a programming language I had never used before.
I wrote this program to explore, document, and demonstrate the [Zig](https://ziglang.org/) programming language.

https://github.com/user-attachments/assets/e208b5a3-5f2f-4d24-9858-e7ebf7938ae7

# Build & Run
Using [`zvm`](https://www.zvm.app/), Zig version `0.16.0`
```
git clone https://github.com/sean-o-reilly/zig101 && cd zig101
zvm run 0.16.0 build run
```
